--- layout: post title: Control the order of resource management in Puppet description: How to control the order of resource management in Puppet tags: - puppet - automation - infrastructure --- A common heard complaint when using Puppet for managing resources is that it is difficult to manage the order of how resources should be executed.
The order of resources in a Puppet manifest does not matter. Puppet assumes that most resources are not related to each other and will manage the resources in whatever order is most efficient
This means that you'll need to explicitly define relationships between resource to run them in a specific order. Relationships can be declared by:
  • Relationship metaparameters
  • Chaining arrows
  • The require function
